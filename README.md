##VIM安装步骤
1. dot文件
```
git clone https://github.com/haharazer/dotfile/blob/master/vimrc ~/.vimrc
git clone https://github.com/haharazer/dotfile/blob/master/vim ~/.vim
```
2. 安装插件
```
git clone https://github.com/gmarik/vundle.git ~/.vim/bundle/vundle
```
3. 打开wim
```
:BundleInstall
```
